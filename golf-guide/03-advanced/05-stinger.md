# ⚡ Stinger 壓低彈道技術

## 概覽 (Overview)

Stinger 是由 Tiger Woods 在職業生涯中發揚光大的招牌技術，以一號木桿或長鐵桿打出超低穿刺彈道。這種球路因為彈道極低、側旋極少，在強風中幾乎不受影響，同時提供極高的方向精確度。Stinger 被許多人認為是現代高爾夫中最令人印象深刻的球技之一。

---

## 何時使用 (When to Use)

- 💨 強烈逆風或側風時（Stinger 的強項）
- 🌲 需要極低彈道穿越樹木
- 🎯 需要最高精確度的球道擊球
- ⛳ 狹窄球道需要保持球路在球道中央

---

## 技術分解 (Technique Breakdown)

### 1. 準備動作 (Setup)

Stinger 通常使用**一號木桿或長鐵桿（2-3號鐵）**：

**關鍵設置差異：**
- 球位比正常驅動桿更靠後（站距中心到略前於中心）
- 球座非常低（幾乎齊平地面，使用一號木桿時）
- 手部前壓顯著

---

### 2. 握桿 (Grip)

- 比正常握桿稍短（Choke Down 約1英寸）
- 握力偏緊（增加控制）
- 確保手部前壓姿勢穩定

---

### 3. 站姿 (Stance)

- 腳距正常
- 重心略偏前腳（約55%）
- 脊椎角度保持正常，不過度前傾

---

### 4. 揮桿路徑 (Swing Path)

Stinger 的揮桿與Punch Shot類似，但使用更長的球桿：

```
Stinger 揮桿特點：
- 上桿：約四分之三位置
- 轉速：正常（不刻意減速）
- 下桿：積極、向下壓
- 收桿：低位壓制收桿（最關鍵）
```

**下桿的關鍵感覺：**
> 感覺像是「用球桿桿頭把球釘在地上」——儘管球會飛出，但這個「釘住」的意念讓您保持正確的擊球角度。

---

### 5. 擊球點 (Impact Position)

Stinger 的擊球點是所有球路中最極端的「去除Loft（Deloft）」狀態：

```
一號木桿正常 Loft = 10.5°
Stinger 去除 Loft 後 ≈ 6-8°（如同低Loft球桿）
結果：極低彈道，少量後旋，最大穿刺力
```

**擊球時的身體狀態：**
- 重心幾乎全在前腳
- 臀部已大幅前移（Hip Thrust）
- 手部位置顯著前於桿頭

---

### 6. 收桿 (Follow Through)

這是 Stinger 最具識別度的動作：

```
正常收桿：           Stinger 收桿：
   桿頭↑                桿頭→→（指向目標）
    |                     |
  手臂高舉              手臂保持低位
```

**執行低收桿：**
1. 擊球後，有意識地「鎖住」收桿高度
2. 右臂（後臂）不折疊，保持伸展
3. 桿頭在收桿後指向目標方向
4. 身體轉動但手臂不上舉

> 💡 Tiger Woods 的 Stinger 收桿時，桿頭幾乎與腰部齊高，看起來像是揮桿在中途「定格」。

## Stinger 的彈道分析

```
正常一號木桿：     ___
彈道：           /   ‾‾‾\_____

Stinger：       ___________
彈道：         /            \____

差異：
- Stinger 彈道最高點更低（約25-30碼 vs 40-45碼）
- 空中飛行距離略短，但滾動更多
- 總距離相近（風況好時）
- 強風中 Stinger 顯著優於正常擊球
```

## 常見錯誤 (Common Mistakes)

| 錯誤 | 原因 | 修正方法 |
|------|------|----------|
| 球彈道仍然偏高 | 收桿高位，去除Loft不足 | 更激烈地執行低收桿 |
| 距離損失太多 | 揮桿速度下降 | 保持積極下桿，只改變收桿 |
| 球路不直 | 球位不正確 | 調整球位到站距中心偏前 |
| 打薄球 | 球位太靠後或手部前壓過度 | 調整球位，減少前壓程度 |

## 練習訓練 (Practice Drills)

### 訓練一：低網格練習
1. 設置一個低矮的網格（約3英尺高，距球15英尺前方）
2. 練習讓球從網格下方穿越
3. 這個視覺目標強迫您打出低彈道
4. 每次練習20球

### 訓練二：「指向目標」收桿
1. 做空揮桿
2. 擊球後，讓桿頭在腰部高度「凍結」，指向目標
3. 保持此姿勢5秒
4. 確認手臂延伸，桿頭指向正確方向
5. 重複20次後加球練習

## 進階提示 (Pro Tips)

> 💡 **Tiger Woods 在 2006 年英國公開賽（皇家霍爾克）的分析**：Stinger 是他在 Hoylake 鏈接球場的主要武器，他一整週幾乎不使用一號木桿，而是以長鐵桿 Stinger 控制球路，最終奪冠。這次表演成為球場管理的經典案例。
