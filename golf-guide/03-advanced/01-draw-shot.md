# 🔄 右曲球技術 (Draw Shot)

## 概覽 (Overview)

右曲球（Draw）是指球從目標左方飛出，向右彎曲落在目標附近的球路（右手球手）。Draw 是許多職業球手最偏愛的球路，因為它能產生更低的彈道、更多的前旋、更遠的落地距離。掌握 Draw 技術，是從中級球手邁向進階的重要里程碑。

---

## 何時使用 (When to Use)

- ↪️ 球道向右轉彎（Right Dogleg）的洞
- 💨 左側強風（抵抗左側風）
- 📏 需要比平時多10-15碼距離時
- 🌲 障礙物在球道左側，需要球從左方繞過

---

## 技術分解 (Technique Breakdown)

### 1. 準備動作 (Setup)

> 📌 **本文以右打者（Right-Handed Golfer）為基準。左打者請參閱[左打者版本](./left-handed-reference.md)。**

Draw 的核心是「由內向外」的揮桿路徑（In-to-Out Swing Path）：

```
右打者（Right-Handed Golfer）設置 - 鳥瞰圖：

  內側（In，身體側）←         → 外側（Out）
  （球手站在此側）

                  [目標]
                     ↑ ← 桿面對準方向
                     │  目標線
              ───────[●]───────  球的位置
                  ╱  │
                ╱    │ ← 身體對準線（偏目標右方 10-15 碼）
              ╱      │
            ↙[球手]  │ （閉合式站姿 Closed Stance）

  • 桿面（Club Face）：↑ 對準真實目標
  • 腳線/身體（Body）：↗ 偏右 10-15 碼（閉合式站姿）
```

**設置步驟：**
1. 確認目標線方向
2. 桿面對準**真正目標**（不偏移）
3. 身體（腳、膝、臀、肩）對準目標**右側** 10-15 碼
4. 保持身體對齊與桿面對齊的差異

> 🎯 **關鍵概念**：桿面對準真實目標，身體對準目標右側。這個差異產生由內向外的揮桿路徑，創造右旋（Draw Spin）。

---

### 2. 握桿 (Grip)

Draw 握桿需要「強握法（Strong Grip）」：

- 左手：拳頭關節**向右旋轉**，可見3個指關節（一般為2個）
- 右手：相應跟隨左手旋轉
- 「V」字形指向右肩偏右

**強握法的效果：**
強握法讓手腕在揮桿中更容易在衝擊前**關閉桿面**，產生右旋的球路。

---

### 3. 站姿 (Stance)

- **腳線方向**：目標右側（閉合式站姿 Closed Stance）
- **球位**：比平時稍靠後（促進由內向外路徑）
- **重心**：略偏後腳（允許更長的由內向外路徑）

---

### 4. 揮桿路徑 (Swing Path) ⭐ 最關鍵

Draw 的揮桿路徑：**由內向外（In-to-Out）**

```
右打者（Right-Handed Golfer）揮桿路徑 - 鳥瞰圖：

  內側（In，身體側）←   目標線↑   → 外側（Out）
  （球手站在左側）            │            （遠離身體）
                              │
    上桿路徑 ╲                │                  （內側起桿）╲           │
                   ╲          │
                    ╲         │
                     ╲        │
                      ╲       │
                       [●] ← 擊球點（在目標線上）
                              │  ╱ 收桿路徑
                              │╱   （外側延伸）
                            ╱ │
                          ╱   │
                        ╱     │

  揮桿路徑：由左側（內側）→ 右側（外側）= In-to-Out
  球路：初始偏左起飛 → 右旋（Draw Spin）→ 向右彎回目標
```

> 📌 **右打者說明**：右打者站在目標線**左側**（內側）。  
> 「由內向外（In-to-Out）」意味著在擊球時，桿頭從身體側（左/內側）穿越目標線至右側（外側）。

**實現由內向外路徑的方法：**
1. 想像球道上有一個「象限」，從球的左後方向右前方（目標方向偏右）揮桿
2. 感覺右肘貼近身體下桿
3. 桿頭從低內側進入擊球區，然後向右上方延伸

---

### 5. 擊球點 (Impact Position)

Draw 的擊球點特徵：
- 桿面**略微關閉**（相對於揮桿路徑）
- 手部在桿頭前方（前壓）
- 重心前移，臀部已轉向目標

**D平面理論應用：**
```
揮桿路徑：目標右方 5 度（In-to-Out）
桿面方向：目標方向（或略偏右1-2度）
結果：球向右曲（右旋 Draw Spin）

Draw 幅度 = 揮桿路徑角度 - 桿面角度
若路徑右偏5度，桿面右偏2度 = 3度的Draw弧度
```

---

### 6. 收桿 (Follow Through)

- 右手（後手）在衝擊後「翻越」左手
- 桿頭指向目標右方（延伸路徑方向）
- 右肩充分轉向目標

---

## 📊 Draw 與 Fade 的距離比較

| 球路 | 彈道 | 落地後滾動 | 總距離 |
|------|------|---------|--------|
| Draw | 低至中 | 多（前旋）| +10-15碼 |
| Straight | 標準 | 標準 | 基準 |
| Fade | 中至高 | 少（後旋）| -5-10碼 |

---

## 常見錯誤 (Common Mistakes)

| 錯誤 | 原因 | 修正方法 |
|------|------|----------|
| Hook（過度右曲）| 桿面太關閉 | 調整到弱握法，或確認桿面對準目標 |
| 球反而向左（Push-Draw）| 身體對齊偏差 | 重新確認桿面與身體的對齊差異 |
| 球路直線（無曲球）| 揮桿路徑不夠In-to-Out | 加大身體與目標的對齊差異 |
| 打薄（Thin）| 球位太靠前 | 將球位稍微向後調整 |

## 練習訓練 (Practice Drills)

### 訓練一：對齊桿練習
1. 放置兩根對齊桿在地面
2. 一根對準真正目標（代表桿面方向）
3. 另一根偏向右方10度（代表身體對齊）
4. 按照此設置練習50球，培養肌肉記憶

### 訓練二：彈彈手腕練習
1. 做短距離（50碼）的半揮桿
2. 意識地讓右手在衝擊後翻越左手
3. 感受「關閉桿面」的感覺
4. 確認球路是否有右曲趨勢，逐漸加大揮桿

### 訓練三：球座 Draw 練習
1. 使用1號木桿，球座稍低
2. 設置閉合式站姿，桿面對準目標
3. 以慢速揮桿感受路徑
4. 逐漸增加揮桿速度

## 進階提示 (Pro Tips)

> 💡 **Tiger Woods 談 Draw**：「我的Draw是透過更強的下肢驅動和臀部旋轉產生的。當臀部更快地旋轉，手部稍微落後，自然產生桿面關閉的效果。」

> 💡 對於初學Draw的球手：先在**短鐵桿**（8號、9號鐵）練習Draw的感覺，短鐵桿的反饋更直接，容易調整。一旦掌握感覺，再移到長鐵桿和木桿。
