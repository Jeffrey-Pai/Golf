# 🔄 左打者參考指南（Left-Handed Golfer Reference）

## 概覽 (Overview)

本文是專為**左打者（Left-Handed Golfer）**所整理的 Draw 和 Fade 技術參考。  
由於高爾夫教材多以右打者（Right-Handed Golfer）為基準，左打者需要將所有方向說明**左右對調**。

> 📌 **右打者請參閱：**
> - [右曲球 Draw（右打者版）](./01-draw-shot.md)
> - [左曲球 Fade（右打者版）](./02-fade-shot.md)

---

## 鳥瞰圖方向說明（重要）

在鳥瞰圖（目標朝上）中，不同慣用手的球手站位如下：

```
鳥瞰圖方向示意（目標在上方）：

              [目標]
                 ↑
                 │  目標線
          ───────[●]───────
                 │
   右打者站在左側 │ 左打者站在右側
   （內側 = 左）  │ （內側 = 右）
```

| 說明項目 | 右打者 | 左打者 |
|---------|--------|--------|
| 球手站位 | 目標線**左側** | 目標線**右側** |
| 內側（Inside） | 左側 | 右側 |
| 外側（Outside） | 右側 | 左側 |
| Draw 揮桿路徑 | 由左→右（In-to-Out） | 由右→左（In-to-Out） |
| Fade 揮桿路徑 | 由右→左（Out-to-In） | 由左→右（Out-to-In） |
| Draw 閉合站姿 | 身體偏目標**右方** | 身體偏目標**左方** |
| Fade 開放站姿 | 身體偏目標**左方** | 身體偏目標**右方** |
| 強握法（Draw） | 左手指關節向右旋轉 | 右手指關節向左旋轉 |
| 弱握法（Fade） | 左手指關節向左旋轉 | 右手指關節向右旋轉 |

## 左打者 Draw（右曲球）

### 什麼是左打者的 Draw？

左打者的 Draw 是指球從目標**右方**飛出，向**左方**彎曲落在目標附近的球路。

### 設置圖

```
左打者（Left-Handed Golfer）Draw 設置 - 鳥瞰圖：

  外側（Out）←         → 內側（In，身體側）
                          （球手站在此側）

                  [目標]
                     ↑ ← 桿面對準方向
                     │  目標線
              ───────[●]───────  球的位置
                     │  ╲
                     │    ╲ ← 身體對準線（偏目標左方 10-15 碼）
                     │      ╲
                     │  [球手]↘（閉合式站姿，站在目標線右側）

  • 桿面（Club Face）：↑ 對準真實目標
  • 腳線/身體（Body）：↙ 偏左 10-15 碼（閉合式站姿）
```

### 揮桿路徑

```
左打者 Draw 揮桿路徑 - 鳥瞰圖：

  外側（Out）←   目標線↑   → 內側（In，身體側）
  （遠離身體）         │         （球手站在右側）
                       │
  上桿路徑              │               ╱ 收桿路徑
  （內側起桿）           │             ╱（外側延伸）
                        │           ╱
                        │         ╱
                        │       ╱
                        │     ╱
                       [●] ← 擊球點（在目標線上）
                   ╱    │
                 ╱      │
               ╱        │

  揮桿路徑：由右側（內側）→ 左側（外側）= In-to-Out（左打者）
  球路：初始偏右起飛 → 右旋（Draw Spin）→ 向左彎回目標
```

### 握桿

- **右手**（主力手/Lead Hand）指關節**向左旋轉**，可見 2.5–3 個指關節
- 「V」字形指向左肩偏左
- 這是左打者的**強握法（Strong Grip）**

### 站姿

- **腳線方向**：目標左側（閉合式站姿 Closed Stance）
- **球位**：比平時稍靠後（促進 In-to-Out 路徑）

## 左打者 Fade（左曲球）

### 什麼是左打者的 Fade？

左打者的 Fade 是指球從目標**左方**飛出，向**右方**彎曲落在目標附近的球路。

### 設置圖

```
左打者（Left-Handed Golfer）Fade 設置 - 鳥瞰圖：

  外側（Out）←         → 內側（In，身體側）
                          （球手站在此側）

                  [目標]
                     ↑ ← 桿面對準方向
                     │  目標線
              ───────[●]───────  球的位置
                  ╱  │
                ╱    │ ← 身體對準線（偏目標右方 10-15 碼）
              ╱      │
         [球手]↙     │（開放式站姿，站在目標線右側）

  • 桿面（Club Face）：↑ 對準真實目標
  • 腳線/身體（Body）：↗ 偏右 10-15 碼（開放式站姿）
```

### 揮桿路徑

```
左打者 Fade 揮桿路徑 - 鳥瞰圖：

  外側（Out）←   目標線↑   → 內側（In，身體側）
  （遠離身體）         │         （球手站在右側）
                       │
  上桿路徑 ╲           │
  （外側起桿）  ╲       │
                 ╲     │
                  ╲    │
                   ╲   │
                    ╲  │
                     [●] ← 擊球點（在目標線上）
                       │  ╱ 收桿路徑
                       │╱   （內側延伸）
                     ╱ │
                   ╱   │
                 ╱     │

  揮桿路徑：由左側（外側）→ 右側（內側）= Out-to-In（左打者）
  球路：初始偏左起飛 → 左旋（Fade Spin）→ 向右彎回目標
```

### 握桿

- **右手**（主力手/Lead Hand）指關節**向右旋轉**，只見 1–1.5 個指關節
- 「V」字形指向下巴或右肩
- 這是左打者的**弱握法（Weak Grip）**

### 站姿

- **腳線方向**：目標右側（開放式站姿 Open Stance）
- **球位**：比平時稍靠前（促進 Out-to-In 路徑）

## 快速對照：左打者 vs. 右打者

| 球路 | 右打者球路方向 | 左打者球路方向 |
|------|--------------|--------------|
| Draw | 從左起飛，向右彎 | 從右起飛，向左彎 |
| Fade | 從右起飛，向左彎 | 從左起飛，向右彎 |
| Hook（失控Draw）| 嚴重向右偏 | 嚴重向左偏 |
| Slice（失控Fade）| 嚴重向左偏 | 嚴重向右偏 |

> 💡 **記憶口訣（左打者）**：把所有右打者教材的「左」「右」互換即可。
