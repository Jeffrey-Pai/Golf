# 🏌️ 鐵桿技術 (Iron Play)

## 概覽 (Overview)

鐵桿是高爾夫球袋中最常用的球桿類型，從長距離的4號鐵到精準的9號鐵，各有其專屬用途。掌握鐵桿技術是成為全面球手的關鍵，尤其是在球道上穩定地打向果嶺，是降低差點的核心能力。鐵桿與木桿最大的差異在於需要「向下打」（Downward Strike），讓桿面先接觸球，再帶起草皮。

---

## 何時使用 (When to Use)

- 🎯 球道上距離旗桿 80-200 碼的擊球
- ⛳ Par 3 開球（選擇適當號數的鐵桿）
- 🌿 長草區逃脫（使用較短號碼鐵桿）
- 📐 需要精準落點的情況

---

## 技術分解 (Technique Breakdown)

### 1. 準備動作 (Setup)

鐵桿的設置與木桿有所不同：

**球的位置：**
- 長鐵桿（4-5號）：站距中心偏前一個球
- 中鐵桿（6-7號）：站距正中心
- 短鐵桿（8-9號、PW）：站距中心偏後一個球

**身體設置：**
- 腳距略窄於木桿（肩寬即可）
- 手部（grip end）略前於桿頭，形成「前壓（Forward Press）」姿勢
- 重心略偏後腳（55:45 後腳:前腳）

---

### 2. 握桿 (Grip)

- 使用標準的互扣式或交叉式握法
- 握力稍緊於木桿（提升控制感）
- 確保左手掌根施加向下的壓力

---

### 3. 站姿 (Stance)

- **腳距**：肩寬或略窄
- **膝蓋**：適度彎曲
- **背部**：直立（與地面約30-40度角）
- **眼睛**：直視球的後方

---

### 4. 揮桿路徑 (Swing Path)

鐵桿揮桿的核心概念：**向下擊球（Descending Blow）**

```
木桿揮桿弧度：
       ← 上桿    下桿 →
         \         /
          \       /
           [底部] ← 以底部為最低點，掃球
            球
            

鐵桿揮桿弧度：
       ← 上桿    下桿 →
         \         /
          \       /
           [球] ← 在到達最低點前先打到球
            ↓
           [最低點在球前方，帶起草皮]
```

**重點流程：**
1. 起桿時肩膀旋轉，不過度依賴手腕
2. 上桿到四分之三位置（不需要完整上桿）
3. 下桿以臀部帶動，重心前移
4. 擊球後繼續「壓桿」穿過球，不要急著收桿

---

### 5. 擊球點 (Impact Position)

**理想鐵桿擊球位置：**
- 手部略前於桿頭（前壓 Forward Press）
- 重心 60-70% 在前腳
- 臀部已轉向目標
- 桿面「捏住」球後帶起草皮（Divot 應在球的前方）

**草皮刀痕（Divot）的解讀：**
| 刀痕位置 | 含義 |
|---------|------|
| 球的前方（目標方向）| ✅ 正確的下降擊球 |
| 球的後方 | ❌ 打胖球（Fat Shot），先打地再打球 |
| 無刀痕 | ❌ 掃球或打薄球（Thin/Skulled Shot）|

---

### 6. 收桿 (Follow Through)

- 手臂充分延伸
- 身體充分轉向目標
- 右肩指向目標（右手球手）
- 右腳腳跟離地

---

## 常見錯誤 (Common Mistakes)

| 錯誤 | 原因 | 修正方法 |
|------|------|----------|
| 打胖球（Fat Shot）| 底部最低點在球後方 | 確認重心前移，手部前壓 |
| 打薄球（Thin）| 掃球或身體起伏 | 保持固定的脊椎角度 |
| 球路偏左（Pull）| 揮桿路徑由外向內（Out-to-In）| 練習由內而外的揮桿路徑 |
| 球路偏右（Push）| 揮桿路徑由內向外（In-to-Out）| 確認目標線對齊 |
| 距離不穩定 | 每次擊球點不一致 | 專注於每次重複相同的設置 |

---

## 練習訓練 (Practice Drills)

### 訓練一：球後放置障礙物
1. 在球後方5公分放一個小物件（球座、橡皮擦）
2. 練習擊球而不碰到障礙物
3. 強迫下降擊球習慣
4. 每次練習20球

### 訓練二：刀痕位置確認
1. 在草地練習場（非人工草皮）打球
2. 每打一球後，檢查草皮刀痕位置
3. 目標：刀痕始終在球的前方（目標線方向）
4. 每次練習50球，記錄刀痕位置比例

### 訓練三：半揮桿鐵桿控制
1. 以7號鐵桿做半揮桿
2. 目標：固定落點在50碼
3. 感受鐵桿「捏球」的質感
4. 一旦穩定後，逐漸增加揮桿幅度

## 進階提示 (Pro Tips)

> 💡 **職業球手的鐵桿秘訣**：「我從不嘗試擊遠。我的鐵桿技術是讓桿頭做功，我只是確保在正確的時間以正確的角度擊中球。」——Tiger Woods

> 💡 每支鐵桿的標準距離是您選桿的基礎。建議在練習場測量您每支鐵桿的**實際飛行距離**（非理想距離），而非依賴教科書數字。
