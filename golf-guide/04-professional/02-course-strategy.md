# 🗺️ 職業球場策略 (Course Strategy)

## 概覽 (Overview)

職業球場策略超越了基本的「安全打法」，進入到主動進攻與精算風險的藝術層面。職業球手在每一輪比賽中都制定詳細的球場計劃，識別可進攻的旗桿位置、需要保守的洞，並根據比賽態勢動態調整策略。這種系統性思維是職業球手與業餘球手差距最大的地方之一。

---

## 何時使用 (When to Use)

- 🏆 任何正式比賽前和進行中
- 📋 賽前勘察球場（Practice Round）
- 🧮 每洞的策略制定
- 🔄 根據比賽態勢調整策略時

---

## 職業球場策略框架

### 第一層：全輪視角（Round View）

```
比賽前問題：
1. 哪些洞是「可攻」的洞（Birdie Opportunities）？
2. 哪些洞是「保par」洞（Par Save Holes）？
3. 哪些洞是「避免雙柏忌」洞（Double Bogey Avoidance）？
4. 我今天的狀態如何？需要調整策略嗎？
```

**攻守比例建議：**
| 比賽狀況 | 進攻旗桿 | 保守打法 |
|---------|---------|---------|
| 領先時 | 40% | 60% |
| 落後需追分 | 70% | 30% |
| 最後輪，差距大 | 80% | 20% |
| 平局進入最後洞 | 依情況判斷 | 依情況判斷 |

---

### 第二層：每洞策略（Hole-by-Hole Planning）

**旗桿位置分類系統：**

```
果嶺分為9個區域（類似九宮格）：

左後 | 中後 | 右後
—————+——————+—————
左中 | 中心 | 右中
—————+——————+—————
左前 | 中前 | 右前
```

| 旗桿位置 | 攻擊策略 |
|---------|---------|
| 中心/中前 | 可直接攻旗，風險最小 |
| 邊角位置（左後/右後）| 評估危險，通常打向安全側 |
| 靠近障礙（水/沙坑）| 打向果嶺安全區域，不冒險 |

---

### 第三層：發球台策略（Tee Shot Strategy）

**職業球手的發球台思維：**

1. **逆向思考（Backward Planning）**：
   先想「我希望第二桿或第三桿從哪個位置打？」，然後倒推開球目標。

2. **危險識別**：
   ```
   球道左側危險（OB/水） → 開球瞄準左側，寧可進長草右側
   球道右側危險（沙坑）→ 開球瞄準右側，讓沙坑在身體另一側
   ```

3. **球道最佳進攻角度**：
   某些旗桿位置從球道特定側進攻角度更好：
   ```
   旗桿在果嶺右側 → 從球道左側進攻角度更好
   旗桿在果嶺左側 → 從球道右側進攻角度更好
   ```

---

### 第四層：果嶺閱讀策略（Green Reading Strategy）

**職業推桿策略（AimPoint 系統）：**

```
推桿線計算要素：
1. 果嶺整體坡度方向
2. 坡度百分比（通過站立感受）
3. 草紋方向（順紋/逆紋）
4. 球速快慢
5. 是否有額外斜坡影響

AimPoint 估算：
- 1格：約1杯（球洞直徑）的偏移量
- 2格：約2杯的偏移量
```

## 技術分解 (Technique Breakdown)

### 1. 準備動作 - 賽前勘察 (Pre-round Reconnaissance)

職業球手在練習輪（Practice Round）中的工作：
- 📏 **測量距離**：確認各標記的實際距離
- 🌿 **草紋觀察**：記錄果嶺草紋方向
- ⛱️ **障礙記錄**：記錄沙坑、水障礙的確切位置和距離
- 📊 **旗桿位置思考**：針對可能的旗桿位置制定進攻策略

---

### 2. 握桿 (Grip) - 在策略中的角色

在職業策略中，握法需要配合選定的球路：
- 選擇了Draw策略 → 確認強握法在手
- 選擇了Fade策略 → 確認弱握法在手
- 保守策略 → 標準握法

---

### 3. 站姿 (Stance) - 戰略站姿

**戰略確認清單（每桿）：**
```
✓ 確認距離（測距儀/距離標記）
✓ 確認風況（方向、強度）
✓ 確認旗桿位置（難/易）
✓ 決定目標（旗桿 or 果嶺中央）
✓ 選擇球路（直/Draw/Fade）
✓ 選擇球桿
✓ 設置對應站姿
✓ 執行
```

---

### 4. 比賽末段策略（Late-Round Strategy）

**最後三洞：**
- 領先：每桿都打安全區域，避免高風險球路
- 落後：計算需要幾個小鳥，決定是否需要冒更大風險
- 平局：保持冷靜，打自己的球，不要試圖「做太多」

---

## 常見錯誤 (Common Mistakes)

| 錯誤 | 原因 | 修正方法 |
|------|------|----------|
| 每洞都攻旗 | 沒有系統策略 | 建立旗桿位置評估系統 |
| 壞球後改變策略 | 情緒化反應 | 遵守賽前制定的計劃 |
| 只考慮距離，不考慮進攻角度 | 一維思考 | 從目標到發球台逆向思考 |
| 比賽末段改變比賽策略 | 壓力影響 | 末段繼續執行整輪一貫策略 |

---

## 練習訓練 (Practice Drills)

### 訓練一：策略日誌
1. 每次下場前，為每個洞寫下計劃（可縮寫）
2. 下場後，對比計劃與執行，標注哪些洞策略有效、哪些需要調整
3. 累積5-10輪的策略日誌，識別系統性問題

### 訓練二：模擬比賽壓力下策略
1. 在練習場，假設自己在比賽的第17洞，差距一桿領先
2. 制定保守策略，練習執行
3. 再假設差距一桿落後，制定進攻策略
4. 感受策略轉換對擊球心理的影響

## 進階提示 (Pro Tips)

> 💡 **Jack Nicklaus 的策略思維**：「我從不打一桿是沒有完整計劃的。在我擊球之前，我已經知道：球路是什麼、目標在哪裡、如果打歪了哪個方向最安全。這種系統性的思維，讓我在職業生涯中避免了無數潛在的災難。」

> 💡 **記住「30碼圈」策略**：如果每桿都能讓自己下一桿在30碼圈內，您的短打技術就能把損失降到最低，而不是試圖每桿都完美。
