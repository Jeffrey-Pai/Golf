# 🏆 比賽準備 (Tournament Preparation)

## 概覽 (Overview)

系統性的比賽準備是職業球手與業餘球手的顯著差距之一。職業球手在正式比賽前有詳細的準備流程，從賽前一週的球場勘察，到比賽當天的例行程序，每個環節都有計劃。本指南將職業比賽準備流程系統化，讓業餘球手也能借鑒並建立自己的比賽準備系統。

---

## 何時使用 (When to Use)

- 🏆 任何正式高爾夫比賽前
- 🎯 需要發揮最佳水準的重要輪次
- 📅 重要業餘聯賽或邀請賽
- 🎖️ 差點記錄輪次（Handicap Rounds）

---

## 賽前一週準備計劃

### 7天前
- 📊 **訂定目標桿數**：根據球場難度和個人水準，設定現實目標
- 🏋️ **完成最後一次高強度訓練**：比賽週不宜進行高強度體能訓練
- 📝 **研究球場資訊**：收集球場資料、坡度圖、旗桿位置歷史數據

### 5天前
- ⛳ **練習輪（Practice Round）**：
  - 記錄每個旗桿位置的最佳進攻角度
  - 測量關鍵距離（超越距離標記的精確位置）
  - 觀察果嶺草紋方向和速度
  - 確認危險區域的確切位置

### 3天前
- 🎯 **針對性練習**：
  - 加強對特定球場最需要的技術（如沙坑多的球場加強沙坑練習）
  - 短打特訓（推桿、切球）
  - 模擬比賽場景的心理練習

### 1天前（賽前日）
- 🏌️ **輕量練習**：只需要感覺良好，不需要解決技術問題
- 🛌 **充分睡眠**：比賽前一晚的睡眠至關重要
- 🥗 **飲食管理**：避免新食物，保持正常飲食習慣
- 🗂️ **確認裝備**：
  ```
  裝備確認清單：
  ✓ 球桿（14支以內）
  ✓ 足夠數量的球
  ✓ 手套（至少2隻備用）
  ✓ 球座
  ✓ 雨衣（天氣預報確認）
  ✓ 飲水和食物
  ✓ 測距儀（確認電池）
  ✓ 計分卡和筆
  ✓ 高爾夫鞋（確認釘子完好）
  ```

## 比賽當天準備

### 出發前（家中）
- ⏰ **提早計劃出發時間**：比報到時間早90分鐘以上到達球場
- 🥗 **適量早餐**：在比賽前2-3小時用餐，避免太飽或太餓

### 到達球場（開球前60分鐘）

```
比賽當天時間表範例（9:00 AM 開球）：

7:30 AM - 到達球場，裝備整理
7:45 AM - 輕伸展（10分鐘）
8:00 AM - 推桿練習（20分鐘）
           目標：感覺果嶺速度，建立信心
8:20 AM - 短打練習（10分鐘）
           切球、沙坑各5球
8:30 AM - 全揮桿熱身（20分鐘）
           從短桿開始，逐漸到長桿
8:50 AM - 一號木桿幾球，確認感覺
8:55 AM - 回到推桿練習，打進幾個短推桿結束
9:00 AM - 比賽開始
```

## 技術分解 (Technique Breakdown)

### 1. 準備動作 (Setup) - 比賽心態設置

**賽前心態建立：**
- 回顧您的「信心成就清單」（以前的好成績、好球）
- 提醒自己比賽的目標是**過程執行**，而非結果
- 接受比賽中會有失誤，計劃如何應對

---

### 2. 握桿 (Grip) - 一致性確認

比賽前，確認您的握桿習慣：
- 壓力時握桿是否變緊？（常見問題）
- 例行程序中是否有重新確認握力的步驟？

---

### 3. 比賽中的策略執行

**每洞開球前的心理清單：**
```
1. 這個洞我的計劃是什麼？（賽前已制定）
2. 今天的狀態是否需要調整計劃？
3. 風況如何，需要調整嗎？
4. 我的目標是進攻還是保守？
5. 執行例行程序，擊球。
```

---

### 4. 揮桿路徑 - 比賽中的技術應對

**「比賽模式」vs「練習模式」的差異：**

| 練習模式 | 比賽模式 |
|---------|---------|
| 思考技術細節 | 信任已建立的技術 |
| 反覆嘗試修正 | 接受當天狀態，充分利用 |
| 追求完美動作 | 追求最優結果 |
| 可以重打 | 一桿定案，接受結果 |

---

### 5. 擊球點 - 比賽節奏管理

**比賽節奏（Pace of Play）管理：**
- 提前準備，不讓節奏打亂心理狀態
- 在等待時：不要過度思考，用輕鬆的方式分心
- 輪到自己時：快速進入例行程序模式

---

### 6. 收桿 - 賽後分析

**賽後分析系統（Shot by Shot Analysis）：**

```
每輪比賽後記錄：
1. 球道命中率（Fairway Hit %）
2. 果嶺命中率（GIR - Green in Regulation %）
3. 每輪推桿數（Putts per Round）
4. 沙坑救球率（Sand Save %）
5. 每輪桿數（Score）

分析重點：
- 哪些洞失桿（Bogey以上）
- 失桿原因（長桿、短打、推桿、罰桿）
- 哪些方面需要重點練習
```

## 常見錯誤 (Common Mistakes)

| 錯誤 | 原因 | 修正方法 |
|------|------|----------|
| 賽前過度練習 | 焦慮導致用練習代替信心 | 賽前日的練習以感覺良好為目的，不解決技術問題 |
| 改變比賽當天例行程序 | 試圖「特別準備」| 維持平時的例行程序，熟悉比安慰更重要 |
| 比賽中嘗試修正技術 | 一個壞球後的恐慌反應 | 比賽中接受當天狀態，賽後才解決技術問題 |
| 與其他選手比較 | 失去焦點 | 只關注自己的記分卡 |
| 第一洞過度謹慎 | 緊張 | 第一洞執行「保守但積極」的計劃，暖身為主 |

## 練習訓練 (Practice Drills)

### 訓練一：模擬比賽輪次
1. 每月至少一次，在練習場模擬完整的18洞比賽
2. 給每個「假設球洞」設定距離和情景
3. 按照比賽例行程序，認真擊打每一桿
4. 記錄「虛擬分數」，模擬比賽壓力

### 訓練二：賽前熱身流程練習
1. 練習並固定您的賽前熱身流程（60分鐘版和30分鐘版）
2. 確保流程讓您感到「就緒」而非「焦慮」
3. 在正式比賽前的多次練習輪中重複此流程

## 進階提示 (Pro Tips)

> 💡 **Arnold Palmer 的比賽哲學**：「在重要比賽前，我什麼特別的事都不做。我的準備就是我平時的準備。因為最好的準備，就是建立到你的準備沒有什麼特別的。」

> 💡 **最重要的賽前建議**：前一晚的睡眠比所有技術準備都重要。一個充分休息的大腦，能做出更好的決策，更有效率地管理情緒。任何技術上的最後修煉，都不值得犧牲睡眠。
