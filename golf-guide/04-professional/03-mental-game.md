# 🧠 心理訓練 (Mental Game)

## 概覽 (Overview)

高爾夫心理訓練（Mental Game）是職業球手最重要卻最少被討論的領域。許多球手技術優秀，但在比賽壓力下表現失常；反之，心理素質強大的球手往往能在技術發揮80%的情況下打出最佳成績。Bob Rotella 博士（世界頂級高爾夫心理教練）曾說：「高爾夫是一項在脖子以上進行的運動。」

---

## 何時使用 (When to Use)

- 🏆 任何比賽場合（業餘到職業）
- 😰 面對壓力推桿時
- 😤 連續失誤後的心理重建
- 🎯 希望更穩定地發揮技術水準時

---

## 核心心理概念

### 1. 過程導向 vs 結果導向

```
結果導向（錯誤）：         過程導向（正確）：
「我必須把這個推桿進洞」    「我要完美執行這個推桿動作」
「我不能打OB」             「我要瞄準球道中央」
「如果我輸了這場比賽...」   「這一桿我要做的是...」
```

過程導向的思維將注意力放在**可控的事**（您的動作和決策），而非**不可控的事**（球的結果、競爭對手的表現）。

---

### 2. 擊球例行程序（Pre-shot Routine）

**目的：** 創造一致的心理-生理狀態，每次擊球前重複相同的「準備儀式」

**設計您的例行程序（應在3-15秒內完成）：**

```
標準例行程序範例：
1. 從球後方確認目標線 [2秒]
2. 深呼吸，放鬆 [1秒]
3. 1-2次「模擬揮桿」（感覺目標距離）[3秒]
4. 走進擊球位置 [2秒]
5. 最後確認對齊 [1秒]
6. 「觸發詞」（如「平穩」「信任」）[0.5秒]
7. 擊球 [瞬間]
```

**重要原則：**
- 例行程序長度固定（壓力時不加長）
- 在例行程序開始後，**不再思考技術**
- 一旦覺得準備不夠，重新開始例行程序，而非勉強擊球

---

### 3. 注意力管理（Focus Management）

**「當下時刻」技術：**

高爾夫中，注意力只能有效存在於**當下這一桿**。過去的失誤和未來的焦慮都是干擾源：

```
無效的注意力狀態：
過去 → 「上個洞那個三推...」
未來 → 「如果這洞打高分就...」

有效的注意力狀態：
當下 → 「這一桿，我要...」
```

**「停、放、轉」技術：**
1. **停**：察覺自己正在進行負面思考
2. **放**：有意識地「放下」這個想法
3. **轉**：將注意力轉回「這一桿的計劃」

---

## 技術分解 (Technique Breakdown)

### 1. 準備動作 (Setup) - 心理設置

擊球前的心理準備：
- 做決定（選桿、球路、目標），**一旦決定，不再猶豫**
- 啟動例行程序
- 在例行程序中，從策略模式切換到「運動員模式」（讓身體執行）

---

### 2. 壓力管理技術

**生理放鬆技術：**
```
4-7-8 呼吸法：
吸氣 4 秒 → 屏住 7 秒 → 呼出 8 秒

效果：快速降低心跳率，減少腎上腺素，讓身體進入放鬆狀態
```

**「擊球後情緒窗口」理論：**
Bob Rotella 建議球手允許自己在擊球後的**10秒內**有情緒反應（無論好壞），然後強制轉換到中性狀態，為下一桿做準備。

---

### 3. 信心建立系統

**正面自我對話（Positive Self-Talk）：**
- 避免：「不要打OB」（大腦不處理「不要」，只看到「OB」）
- 改為：「瞄準球道中央，放鬆揮桿」

**信心銀行（Confidence Bank）：**
每次成功的揮桿、好的決策，都是存入信心銀行的「存款」。建立信心的方法：
- 在練習場刻意製造成功體驗
- 下場前練習短距離推桿（從必進的距離開始，建立信心）
- 賽後記錄每輪的「亮點」，不只關注失誤

---

### 4. 失誤後恢復

**「沙漏恢復法」：**

```
失誤發生
    ↓
允許情緒（10秒）
    ↓
物理動作重置（握緊拳頭再放開，或脫下手套再戴上）
    ↓
走向下一個球位時，開始分析原因（技術性思維）
    ↓
到達球位時，回到過程導向思維
    ↓
執行例行程序，擊球
```

## 常見心理錯誤 (Common Mistakes)

| 錯誤 | 原因 | 修正方法 |
|------|------|----------|
| 壓力下思考技術細節 | 嘗試「管控」揮桿 | 在例行程序後切換到「運動員模式」|
| 與同組球手比較 | 失去焦點 | 只與記分卡（標準桿）比較 |
| 一個壞球影響整輪 | 情緒外溢 | 嚴格執行失誤恢復程序 |
| 過度分析每桿 | 完美主義 | 接受高爾夫的「不完美性」，把重點放在整輪表現 |
| 緊張時例行程序變長 | 用思考代替信任 | 例行程序時間固定，不因壓力延長 |

## 練習訓練 (Practice Drills)

### 訓練一：壓力推桿練習
1. 設置從1碼到5碼的推桿（每隔1碼一個球）
2. 規定：每個都要進才能繼續
3. 如果某個未進，重新從1碼開始
4. 在模擬壓力下執行例行程序

### 訓練二：「下場前」信心建立
1. 下場前，在練習場先打5個「確定能打好」的短打
2. 感受成功的擊球感覺
3. 帶著這種「成功感覺」上場
4. 讓信心先於技術

### 訓練三：注意力轉換練習
1. 練習時，刻意讓自己想「分心的事情」（比賽結果、他人眼光）
2. 練習「停、放、轉」技術，把注意力拉回當下
3. 目標：在10秒內完成注意力轉換
4. 這種主動練習讓比賽時的轉換更自然

## 進階提示 (Pro Tips)

> 💡 **Tiger Woods 的心理技術**：「我從不在球場上想技術問題。在練習場思考技術，在球場上信任技術。如果我在球場上開始想揮桿動作，那是我焦慮的信號，我需要回到例行程序和過程導向。」
