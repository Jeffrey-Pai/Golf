# 🎳 Bump and Run 滾地短打

## 概覽 (Overview)

Bump and Run 是一種讓球在低彈道下落地，然後在地面上滾行到球洞附近的短打技術。這種球技在英國鏈接球場（Links Course）中非常普遍，因為地面通常比較堅硬，球在草地上滾動可以預測。Bump and Run 是高爾夫中最安全、最可靠的近距離短打方式，風險遠低於 Flop Shot，但需要良好的地面條件配合。

---

## 何時使用 (When to Use)

- 🌿 果嶺周邊有大片平緩的草地（沒有障礙）
- 🏌️ 旗桿距離果嶺邊緣較遠（可以利用滾動距離）
- 💨 強風中需要保持球在風的影響以下
- 🌊 英式鏈接球場風格的打法
- 😰 短打壓力大時，比 Flop Shot 更保守的選擇

> 💡 **風的法則**：當風很大時，把球保持低飛並讓它滾行，比試圖打高飛球更可控。這就是 Bump and Run 的最大優勢。

---

## 技術分解 (Technique Breakdown)

### 1. 準備動作 (Setup)

Bump and Run 的設置與切球（Chip Shot）類似但更極端：

```
典型設置：
- 球位：非常靠近後腳
- 重心：70-80% 在前腳
- 手部：顯著前壓
- 選桿：7號鐵到9號鐵（增加滾動比例）
- 握桿：可以採用推桿握法（Putting Grip）
```

**選桿策略：**

| 球桿 | 飛行:滾動 | 最適場景 |
|------|---------|---------|
| 7號鐵 | 1:4 | 旗桿在果嶺遠端，大片滾行空間 |
| 8號鐵 | 1:3 | 中等距離，適中滾行 |
| 9號鐵 | 1:2.5 | 旗桿在果嶺中段 |
| PW | 1:2 | 旗桿在果嶺近端 |

---

### 2. 握桿 (Grip)

**推桿式握法（Putting Grip）的優勢：**
- 完全消除手腕動作（最大化一致性）
- 讓肩膀主導動作，像是大幅推桿
- 適合壓力下的穩定執行

---

### 3. 站姿 (Stance)

- 腳距非常窄（30cm 以內）
- 重心明顯偏前腳
- 略微開放式站姿（方便觀察球路到旗桿的線路）

---

### 4. 揮桿路徑 (Swing Path)

**「大推桿」的思維模式：**

```
推桿動作：
肩膀帶動，手腕固定
小弧度，對稱揮桿

Bump and Run：
相同的「肩膀帶動」原則
但揮桿弧度更大（配合距離）
手腕仍然固定
```

**標準動作：**
1. 肩膀帶動上桿，手腕不折疊
2. 手部前壓，桿頭向下壓球
3. 收桿：對稱於上桿幅度，方向指向目標

---

### 5. 擊球點 (Impact Position)

- 手部明顯前於桿頭（最重要）
- 桿頭接觸球時「向下壓」
- 幾乎不帶起草皮（只想要球在近地面飛出）

---

### 6. 收桿 (Follow Through)

- 與上桿幅度對稱的短收桿
- 不需要高位收桿
- 桿面指向目標方向

---

## 距離控制方法

**「鐘擺法則」：**
將上桿幅度比喻為時鐘時針的位置：

```
8點鐘 → 10碼
7點鐘 → 15碼
6點鐘（垂直地面）→ 20碼

* 不同球桿數字不同，需要個人測量
```

---

## 常見錯誤 (Common Mistakes)

| 錯誤 | 原因 | 修正方法 |
|------|------|----------|
| 手腕翻動 | 試圖「幫助」球飛起 | 使用推桿握法，完全消除手腕動作 |
| 球飛得太高滾不到旗桿 | 球位太靠前或手部前壓不足 | 調整球位到靠後位置 |
| 距離控制差 | 揮桿力道不一致 | 用「鐘擺幅度」控制距離，而非速度 |
| 低估滾動距離 | 不熟悉不同球桿的飛滾比例 | 建立個人的「飛滾比例表」|

---

## 練習訓練 (Practice Drills)

### 訓練一：飛滾比例測量
1. 在果嶺旁標記落球點和旗桿距離
2. 用不同球桿（7鐵、8鐵、9鐵、PW）各打10球
3. 記錄每支球桿的平均飛行距離和滾動距離
4. 建立個人「Bump and Run 距離表」

### 訓練二：無旗桿距離感練習
1. 在練習果嶺，拔掉或忽略旗桿
2. 只設定落球點目標
3. 專注讓球精準落在目標點上
4. 這個練習建立「飛行距離控制感」

---

## 進階提示 (Pro Tips)

> 💡 **Tom Watson 的名言**（五屆英國公開賽冠軍）：「在風中，把球保持在地面是智慧，不是膽怯。Bump and Run 是最老也是最聰明的高爾夫短打技術。」

> 💡 在果嶺很快的條件下（如乾燥夏日），Bump and Run 比 Flop Shot 更可預測——因為滾動距離可以從落球點計算，而 Flop Shot 的落地反彈在快果嶺上很難控制。
