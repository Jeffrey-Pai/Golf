# 📏 長距推桿技術 (Lag Putting)

## 概覽 (Overview)

長距推桿（Lag Putting）是指從20碼或以上距離進行的推桿——目標不是一推進洞，而是將球「lag（拖）」到球洞附近，讓下一個推桿幾乎確定能夠進洞。Lag Putting 是成熟推桿技術的核心，因為長距推桿很少一推進洞，但一個好的 Lag Putt 可以防止三推桿（Three-Putt），而三推桿是業餘球手浪費桿數最多的地方。

---

## 何時使用 (When to Use)

- 📏 任何距離超過15碼的推桿
- 🎯 果嶺速度特別快，需要控制距離
- 🔧 旗桿在果嶺遠端的困難位置
- 🏆 比賽中，保持穩定比冒險更重要時

---

## Lag Putting 的核心哲學

```
短推桿思維：「進洞」
長距推桿思維：「進入3英尺圈（下一推必進）」

業餘球手的錯誤：
長距推桿也試圖進洞
→ 力道不穩定
→ 距離失控
→ 三推桿

正確思維：
長距推桿目標 = 讓下一推不超過3英尺
→ 壓力大幅降低
→ 距離控制更好
→ 避免三推桿
```

---

## 技術分解 (Technique Breakdown)

### 1. 準備動作 (Setup)

**長距推桿的特殊設置：**
- 站距比短推桿略寬（增加穩定性）
- 手部稍微下移（增加長距離的揮桿感）
- 握桿全長（不要縮短握桿）

---

### 2. 握桿 (Grip)

- 使用習慣握法（反扣式或交叉式）
- 握力要輕（比短推桿更輕）
- 允許更自然的肩膀和手臂動作

---

### 3. 站姿 (Stance)

- 稍寬的站距（比短推桿寬約2英寸）
- 確保眼睛仍在球的正上方
- 身體稍微向前傾斜

---

### 4. 揮桿路徑 (Swing Path)

**長距推桿的動作特點：**

```
長距推桿的「鐘擺」更大：

後擺：比短推桿顯著更長
     約為預期推桿距離的1/10（如20碼推桿 = 2碼後擺）

加速節奏：
  前擺加速應均勻，不要突然「打」球
  想像在整個前擺中持續「推」球
  
收桿：
  自然延伸，不要截斷
  收桿長度 = 後擺長度（對稱鐘擺）
```

---

### 5. 擊球點 (Impact Position)

**長距推桿的衝擊感覺：**
- 感覺平滑、均勻的加速
- 不是「打」而是「推」
- 桿頭速度在衝擊點後仍在加速或維持

**「死手（Soft Hands）」技術：**
長距推桿中，稍微更輕的握力讓推桿頭「感受」地面質地，自然調整力道。這是職業推桿手擁有「神手感」的秘密之一。

---

### 6. 收桿 (Follow Through)

- 收桿充分（比短推桿更長）
- 不要截斷收桿（是三推桿的常見原因）
- 眼睛在桿頭到達前方時可以緩慢抬起（不要急著看球）

---

## 距離控制的關鍵因素

### 因素一：果嶺速度（Stimpmeter Reading）

果嶺速度（Stimp）是測量果嶺速度的標準：

| Stimp值 | 速度描述 | 長距推桿調整 |
|---------|---------|------------|
| 7-8 | 慢（公共球場）| 增加力道 |
| 9-10 | 中等（標準球場）| 標準力道 |
| 11-12 | 快（錦標賽果嶺）| 減少力道 |
| 13+ | 非常快（大型賽事）| 大幅減少力道 |

### 因素二：上坡/下坡推桿

```
上坡推桿：需要更多力道
  → 目標讓球到達球洞高出「2-3個球洞高度」
  
下坡推桿：需要更少力道
  → 目標讓球「軟進洞旁」
  → 下坡推桿最難控制，死亡在球洞上方！
```

## 三推桿預防系統

```
Lag Putting 統計目標：

優秀：從20碼內三推桿率 < 5%
一般：三推桿率 5-15%
需要改善：三推桿率 > 15%

如何追蹤：
每輪記錄三推桿次數
除以18洞 = 三推桿率
```

## 常見錯誤 (Common Mistakes)

| 錯誤 | 原因 | 修正方法 |
|------|------|----------|
| 力道過輕（Short Putt）| 害怕球滾過球洞太多 | 記住「永遠要到達球洞」，死球沒有機會進洞 |
| 力道過重（Long Putt）| 揮桿過大，加速過猛 | 用「鐘擺對稱」控制距離，不是速度 |
| 三推桿（Three-Putt）| 距離判斷失誤 | 轉換思維：目標是「3英尺圈」，不是進洞 |
| 讀線不足 | 長距推桿偏移更多，容易低估 | 長距推桿的偏移量是短距的2-3倍 |

## 練習訓練 (Practice Drills)

### 訓練一：三英尺圈練習
1. 在球洞周圍放置4個球，形成3英尺圓圈的邊界
2. 從15-20碼的長距離推桿
3. 目標：讓球停在圓圈內（不一定進洞）
4. 10個球計算成功率（目標：8/10以上）

### 訓練二：「停留區」距離分級
1. 設置3個停留區：5-10碼（A區）、10-15碼（B區）、15-20碼（C區）
2. 從各種距離推桿，目標球停在最近的A區
3. 從遠距，最終讓球停在2-3碼範圍

### 訓練三：速度感知閉眼練習
1. 閉眼推桿，只靠觸感控制力道
2. 睜眼確認球的距離
3. 訓練「手感」而非視覺計算距離
4. 每次練習10次閉眼推桿

## 進階提示 (Pro Tips)

> 💡 **Dave Pelz（短打科學分析師）的數據**：「最好的推桿策略是讓球在球洞後方6-12英寸停下。進洞的機率是讓球在球洞前停下的4倍。力道要『過洞（Past the Hole）』，不要短。」

> 💡 **果嶺速度適應技術**：在練習推桿綠上，先打3個不同距離的lag putts（感覺果嶺速度），然後再打正式推桿。這15-20秒的「感應」練習，讓您的神經系統在比賽前校準好果嶺速度。
