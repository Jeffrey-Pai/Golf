# ⛳ 高爾夫完整學習指南

> 從初學者到職業選手的全方位高爾夫教學文件

---

## 📖 關於本指南

本指南以繁體中文撰寫，涵蓋高爾夫從入門到職業的完整學習路徑。無論您是第一次拿起球桿，還是希望精進技術的進階球友，都能在這裡找到適合您的學習資源。

---

## 🗺️ 學習路徑總覽

```
初學者 → 中級 → 進階 → 職業
  ⬇        ⬇       ⬇       ⬇
第1-2年   第2-3年  第3-5年  5年以上
差點30+   差點20-30 差點10-20 差點10以下
```

## 🚀 建議學習路徑

### 完全初學者
1. 閱讀 [高爾夫基礎介紹](./01-beginner/01-what-is-golf.md)
2. 了解 [球具選擇指南](./01-beginner/02-equipment-guide.md)
3. 學習 [禮儀與規則](./01-beginner/03-golf-etiquette.md)
4. 練習 [基本站姿與握桿](./01-beginner/04-basic-stance.md)
5. 嘗試 [第一次揮桿](./01-beginner/05-first-swing.md)

### 中級進階
1. 精通 [鐵桿技術](./02-intermediate/01-iron-play.md)
2. 強化 [短打技術](./02-intermediate/03-short-game.md)
3. 學習 [球場管理策略](./02-intermediate/05-course-management.md)
4. 進入 [Draw 技術](./03-advanced/01-draw-shot.md)

### 進階精英
1. 掌握所有 [球路塑造技術](./04-professional/01-shot-shaping.md)
2. 精研 [心理訓練](./04-professional/03-mental-game.md)
3. 執行 [體能訓練計劃](./06-fitness-and-health/strength-training/overview.md)
4. 做好 [傷害預防](./06-fitness-and-health/injury-prevention/wrist-care.md)

## 📝 如何使用本指南

- 每個文件包含**概覽、技術分解、常見錯誤、練習訓練**四大區塊
- 文件頂部的 frontmatter 說明**難度等級**與**前置技術**
- 善用文件內的**相關技術連結**串聯學習
- 遇到不熟悉的術語，請參考 [術語表](./07-reference/glossary.md)
- 健身保養請參考 [06 - 健身與保養](./06-fitness-and-health/README.md)
