# 💪 手肘保養 (Elbow Care for Golf)

手肘傷害約佔高爾夫傷害的7-10%，其中業餘球友的比例遠高於職業選手，主要原因是技術不成熟導致的衝擊代償。最常見的兩種手肘傷害——「高爾夫肘」與「網球肘」——名稱容易混淆，但發生位置、機制和處理方式都不同。正確鑑別是有效治療的第一步。

---

## 📖 概覽：高爾夫肘 vs 網球肘

### 🔴 高爾夫肘 (Golfer's Elbow) — 內上髁炎 (Medial Epicondylitis)

| 項目 | 說明 |
|------|------|
| **中文正式名稱** | 肱骨內上髁炎 |
| **英文名稱** | Medial Epicondylitis |
| **疼痛位置** | 手肘**內側**（靠近身體的那一側） |
| **受損結構** | 前臂屈肌群共同起端肌腱（FCR, FCU, PT, PL）在內上髁的附著點 |
| **受傷動作** | 揮桿時過度用力握桿、腕屈曲動作過猛、碰地擊球 |
| **高危打法** | 試圖「抓緊桿頭加速」、低挖擊球 |

### 🔵 網球肘 (Tennis Elbow) — 外上髁炎 (Lateral Epicondylitis)

| 項目 | 說明 |
|------|------|
| **中文正式名稱** | 肱骨外上髁炎 |
| **英文名稱** | Lateral Epicondylitis |
| **疼痛位置** | 手肘**外側**（遠離身體的那一側） |
| **受損結構** | 橈側腕短伸肌（ECRB）在外上髁的附著點 |
| **受傷動作** | 收桿時手腕過度背伸、握桿過緊造成伸肌過載 |
| **高危打法** | 鷹爪式收桿、用手臂揮桿而非身體旋轉 |

---

## 🔍 自我評估：疼痛位置辨別

手肘對準自己，四根手指朝前：

```
        [肱骨]
       /        \
  外上髁           內上髁
(外側/遠身)      (內側/近身)
  網球肘 🎾        高爾夫肘 ⛳
```

**簡易測試**：
- **高爾夫肘測試**：手腕屈曲（手掌朝上握拳）時內側疼痛加劇 → 高爾夫肘
- **網球肘測試**：中指抗阻伸展（手腕背伸出力）時外側疼痛加劇 → 網球肘

---

## 🏌️ 哪些擊球動作加劇哪側

| 擊球情況 | 主要壓力側 |
|----------|------------|
| 碰地 (Fat Shot) / 硬挖 | 內側（高爾夫肘） |
| 薄擊 (Thin Shot) 抽地 | 外側（網球肘） |
| 握桿過緊 | 兩側均受影響 |
| 高速揮桿收桿 | 外側（非主力手） |
| 碰地爆發力射桿 | 內側（主力手） |

## 🛡️ 預防訓練動作

### 高爾夫肘預防（針對內側屈肌）

| 動作 | 說明 | 組數 | 次數 | 頻率 |
|------|------|------|------|------|
| 離心腕屈曲 (Eccentric Wrist Curl) | 前臂置桌，用健側協助舉起，患側緩慢下降（3-5秒） | 3 | 每側15下 | 每週3次 |
| 前臂旋後強化 | 前臂伸直，拇指朝上，緩慢旋後（手掌向上）抵抗彈力帶 | 3 | 每側12下 | 每週3次 |
| 腕屈肌靜態伸展 | 手臂伸直手掌朝上，另一手輕壓手指向下 | 3 | 每側45秒 | 每日 |
| 握力訓練 | 握力器或軟球輕度訓練（不引發疼痛） | 3 | 每側15下 | 每週3次 |

### 網球肘預防（針對外側伸肌）

| 動作 | 說明 | 組數 | 次數 | 頻率 |
|------|------|------|------|------|
| 泰勒扭轉 (Tyler Twist) | 使用Flex Bar橡皮棒，雙手扭轉後放開——最有研究支持的網球肘運動 | 3 | 每側15下 | 每天 |
| 離心腕伸展 (Eccentric Wrist Extension) | 健側協助舉起，患側緩慢下降 | 3 | 每側15下 | 每週3次 |
| 腕伸肌靜態伸展 | 手臂伸直手掌朝下，另一手輕壓手背向下 | 3 | 每側45秒 | 每日 |
| 前臂旋前強化 | 前臂旋前（手掌朝下）抵抗彈力帶 | 3 | 每側12下 | 每週3次 |

### 通用預防（兩側均適用）

| 動作 | 說明 | 組數 | 次數 | 頻率 |
|------|------|------|------|------|
| 米桶訓練 | 手插米桶做全方位阻力訓練 | 3 | 各30秒 | 每週3次 |
| 前臂旋前/旋後 | 前臂置桌，手持輕啞鈴來回旋轉 | 3 | 每側15下 | 每週3次 |
| 手指展開/閉合 | 手戴橡皮筋，用力展開手指 | 3 | 20下 | 每日 |

## 🩺 何時就診物理治療師或醫師

**應在1週內就診：**
- 手肘疼痛超過2週，保守處理（休息、冰敷）無改善
- 疼痛影響日常生活（開門把、握手、提重物）
- 夜間疼痛加劇，影響睡眠

**應立即就診：**
- 手肘突然無法完全伸直
- 手肘有明顯腫脹或瘀青（懷疑撕裂或骨折）
- 手肘彈響後劇烈疼痛

## 🩹 護肘帶推薦與使用方法

### 反制力護肘帶 (Counterforce Brace / Epicondylitis Strap)

**作用原理**：在肌腱附著點下方施加壓力，改變肌肉拉力方向，減少對肌腱起端的牽拉

**佩戴位置**：
- **高爾夫肘**：束帶套在手肘**內側**，距離內上髁約3-4公分的前臂處
- **網球肘**：束帶套在手肘**外側**，距離外上髁約3-4公分的前臂處

**注意事項**：
- 束帶不應過緊（手指應仍可完整屈曲）
- 護肘帶是輔助，不是治療，疼痛持續必須就醫
- 部分研究顯示護肘帶配合離心運動效果最佳

**推薦品牌**：Mueller、Pro-Tec、Bauerfeind 均有成熟產品

---

## 🔄 修改揮桿減少負擔的技術建議

1. **放鬆握桿壓力**：將握桿力量從8/10降到4-5/10，可顯著減少前臂肌腱壓力
2. **以身體帶動桿頭**：減少手臂主動用力，讓旋轉驅動揮桿
3. **避免「手腕折桿」**：後擺時避免過度扣腕
4. **使用合適重量的桿頭**：過重桿頭加重手臂負擔
